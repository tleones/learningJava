1. Qual é o grande benefício de uma Máquina Virtual Java (JVM)?

Executar o código independente do sistema operacional.

Quais tipos de sistemas você pode criar com a linguagem Java?

4. Qual é a diferença entre o executável do Windows (exe) e o executável do mundo Java (Bytecode)?

Os executáveis do mundo Java são portáveis, os do mundo Windows não.

Os executáveis do mundo Windows podem ser executados diretamente no sistema operacional, os do mundo Java precisam da máquina virtual.
